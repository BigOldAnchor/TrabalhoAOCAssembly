### Exercício: Verificação de capicua
Faça um algoritmo que leia um número (1 < número < 10000 e, caso não seja, deve ser novamente solicitado) e caso ele não seja um número palíndromo (capicua), apontar qual o número palíndromo (capicua) mais próximo a anterior a ele.
